All values except from primitive values are objects in javascript. To create an object in javascript one has to have knowledge of variables. Variables being the memory storage for our values. You need to first declare a variable using the var, let or const keywords. Then you have to input the equals operator and calibraces ,which will declare the scope of your object. Objects are created using key and values and are separated using a comma. If you would wish to get an output from your objects, you can do so by using the console.log keyword. You will have to input the variable name and the key you would like to access from the object. With ES6 that has been made easy. By doing that, you would have sucessfully created an object in Javascript.

Q3.

Global scope − it is has a global scope which means it can be accessed anywhere in your JavaScript code.
Local scope − it has a local scope which is only accesible within a function where it is defined. Function parameters are always local to that function.
A lexical scope - it means that a variable defined outside a function can be accessible inside another function defined after the variable declaration.
